AgriGuardian is an AI-powered assistant designed to help farmers with real-time guidance on weather, pests, crop care, and other agricultural concerns, especially in low-connectivity zones.

In a real deployment scenario, this service would integrate with:

  • SMS messaging via AWS SNS/Lambda for farmers with limited connectivity
  • Real IoT sensors placed in farm fields for accurate data
  • Weather forecasting APIs for predictive guidance
  • Historical farming data for personalized advice

Note: This demo uses the OpenRouter API with DeepSeek's AI model and has a limit of 50 requests per day.
